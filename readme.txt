1. OPENING THE DATABASE TABLE

When you open the application for the first time, a folder named "passwords" is created in the root directory 
of the application, and it creates an empty database file with one empty table.
When you open the passwords directory, you will see that there is a file in that directory called "your_passwords_database"
This file contains a table where all password data generated by "Password Generator" is stored.
New data that will be created in the generator will also be added to this file.
You can view your passwords on another page of the application by clicking "Show table >>>>"
You can also view them by opening the "your_passwords_database" file using online viewers.
For example, using https://inloop.github.io/sqlite-viewer/ or https://sqliteonline.com/ .
If the above websites do not work, you need to find any other or install SQLite.


2. USING THE PASSWORD GENERATOR APPLICATION
	2.1 The application has 2 pages: the main page, where you can directly generate passwords with various
	    settings and save them to the database and an additional page where you can see all saved passwords
	    from the database in the form of a table.

	2.2 On the main page you will be able to see 4 input fields, 9 normal buttons and 6 radio buttons.
		2.2.1 The first field is a field in which you can enter the purpose (description) of your password, the content of this field
		      can be anything, but the length of the entered characters cannot exceed 384 characters and the description of the password
		      cannot be repeated in several passwords at the same time and also the description field cannot be empty.
		2.2.2 The second field is a field in which you can enter the length (number of characters) of your password, the content of this field
		      can only be an integer, the value of which is within certain limits, depending on the selected radio button.
		      But the length value cannot exceed the number 384 and be less than or equal to 0. It cannot be empty.
		2.2.3 The third field is a field where you can allow or prevent the generator from generating passwords that contain
		      repeating characters. The values of this field can only be Latin characters "Y" or "y" - repeated
		      characters are allowed, "N" or "n" - repeated characters are not allowed. It should be noted that by banning repeated
		      characters, the maximum possible password length that the generator can generate decreases. And its length will be
		      depend on the radio button you select.
		2.2.4 The fourth field is the field in which the generated password is displayed, but you can also write your own there
		      password, this field must be filled in to record in the database.
		2.2.5 The first button "Generate" generates a password based on the data obtained from the first three fields and records
		      it into the fourth field, but before that clears it
		2.2.6 The second "UA" button changes the language of all elements of the current program page to Ukrainian.
		2.2.7 The third button "☼" or "☾" changes the interface theme of the current page to light and dark respectively.
		2.2.8 The fourth "EN" button changes the language of all elements of the current program page to English.
		2.2.9 The fifth "Copy password" button copies the fourth field (password), if it is not empty, to the clipboard.
		2.2.10 The sixth button "Clear all" clears all four user input fields of characters.
		2.2.11 The seventh button "Write to database" writes the contents of the first, second and fourth fields to the database.
		       That is, all fields except the third must be filled.
		2.2.12 The eighth "Quit" button for terminating the program. Closing the application.
		2.2.13 Ninth button "Show table >>>>" transition to another page of the program.
		2.2.14 The first radio button "All symbols" generates a password with all Latin letters, numbers and special signs.
		       The maximum password length, if repeated characters are prohibited, is 94.
		2.2.15 The second radio button "Only letters" generates a password with only upper and lower case Latin letters.
		       The maximum password length, if repeated characters are prohibited, is 52.
		2.2.16 The third radio button "Only digits" password generation only from numbers 0 to 9 inclusive.
		       The maximum password length, if repeated characters are prohibited, is 10.
		2.2.17 The fourth radio button "Letters & digits" password generation combines the characters from the previous 2 radio buttons.
		       The maximum password length, if repeated characters are prohibited, is 62.
		2.2.18 Fifth radio button "Letters & signs" password generation with all Latin letters and special signs
		       The maximum password length, if repeated characters are prohibited, is 84.
		2.2.19 Fifth radio button "Digits & signs characters" password generation with all numbers and special signs
		       The maximum password length, if repeated characters are prohibited, is 42.

	2.3 On the additional page you will be able to see 8 ordinary buttons and a table in which you can change 2 buttons.
		2.3.1 The first button "UA" changes the language of all elements of the current page of the program to Ukrainian, and also reloads the table.
		2.3.2 The second button "☼" or "☾" changes the interface theme of the current page to light and dark, and also reloads the table.
		2.3.3 The third button "EN" changes the language of all elements of the current page of the program to English, and also reloads the table.
		2.3.4 The fourth button "<<<< Back" returns the user to the main page of the program.
		2.3.5 The fifth "Reload" button reloads the contents of the table.
		2.3.7 The sixth button "Update" offers to make changes to the database if at least one password description value or password value
		      was changed. If no value has been changed, a corresponding message is displayed. If the password description value is repeated 
		      more than 1 time in the "Password description" column, a corresponding message is displayed..
		2.3.8 The seventh button "Delete" deletes the line containing information about the password by the specified unique identifier,
		      that goes in the first column of the table.
		2.3.9 The eighth "Quit" button for terminating the program. Closing the application.

1. ВІДКРИТТЯ ТАБЛИЦІ БАЗИ ДАНИХ

При першому відкриті додатка в кореневому каталозі додатка створюється папка
"passwords", а в ній створюється файл пустої бази даних з однією пустою таблицею.
Відкривши каталог passwords, ви побачите, що у цьому каталозі є файл під назвою "your_passwords_database"
Цей файл містить таблицю, де збережені усі дані про паролі, згенеровані "Password Generator".
Нові дані, які будуть створені в генераторі, також будуть додані в цей файл.
Переглянути свої паролі ви можете на іншій сторінці додатка, натиснувши "Показати таблицю >>>>"
Також ви можете переглянути їх, відкривши файл "your_passwords_database" за допомогою онлайн переглядачів.
Наприклад, за допомогою https://inloop.github.io/sqlite-viewer/ або https://sqliteonline.com/ .
Якщо вищенаведені веб-сайти не працюють, вам потрібно знайти будь-який інший або інсталювати SQLite.


2. ВИКОРИСТАННЯ ДОДАТКА PASSWORD GENERATOR

	2.1 Додаток має 2 сторінки: головну, в якій ви можете безпосередньо генерувати паролі з різноманітними
	    налаштуваннями та записути їх до бази даних і додаткову сторінку, на якій ви зможете побачити усі збережені паролі
	    з бази даних у вигляді таблиці.

	2.2 На головній сторінці ви зможете побачити 4 поля вводу 9 звичайник кнопкок та 6 радіокнопок.
		2.2.1 Перше поле - це поле, в яке ви можете ввести призначення (опис) свого пароля, вміст цього поля
		      може бути будь-яким, але довжина введених символів не може перевищувати 384 символи і опис пароля
		      не може повторюватися в декількох паролів одночасно і також поле опису не може бути пустим.
		2.2.2 Друге поле - це поле, в яке ви можете ввести довжину (кількість символів) свого пароля, вмістом цього поля
		      може бути тільки ціле число, значення якого знаходиться в певних межах, в залежності від вибраної радіокнопки.
	              Але значення довжини не може перевищувати число 384 і бути меншим або рівним за 0. Не може бути пустим.
		2.2.3 Третє поле - це поле, в якому ви можете дозволити або заборонити генератору створювати паролі, в яких є 
		      повторювані символи. Значеннями цього поля можуть бути тільки латинські символи "Y" або "y" - повторювані 
		      символи дозволені,  "N" або "n" - повторювані символи заборонені. Слід зауважити, що заборонивши повторювані 
		      символи, максимально можлива довжина пароля, який може створити генератор, зменшується. І його довжина буде
		      залежати від вибраної вами раідокнопки.
		2.2.4 Четверте поле - це поле, в якому відображається згенерований пароль, але також ви можете записати туди власний
		      пароль, для запису в базу даних це поле має бути обов'язково заповнене.
		2.2.5 Перша кнопка "Згенерувати пароль" генерує пароль на основі даних отриманих з перших трьох полів і записує
		      його в четверту поле, але перед цим очищає його
		2.2.6 Друга кнопка "UA" змінює мову усіх елементів поточної сторінки програми на українську.
		2.2.7 Третя кнопка "☼" або "☾" змінює тему інтерфейсу поточної сторінки на світлу та темну відповідно.
		2.2.8 Четверта кнопка "EN" змінює мову усіх елементів поточної сторінки програми на англійську.
		2.2.9 П'ята кнопка "Скопіювати пароль" копією четвертого поля (пароль), якщо воно не пусте, в буфер обміну.
		2.2.10 Шоста кнопка "Очистити всі поля" очищає від символів усі чотири поля для вводу користувача.
		2.2.11 Сьома кнопка "Запис до бази даних" записує до бази даних вміст першого, друго та четвертого полів.
		       Тобто усі поля крім третього мають бути обов'язково заповнені.
		2.2.12 Восьма кнопка "Вихід з програми" завершення роботи програми. Закриття додатку.
		2.2.13 Дев'ята кнопка "Показати таблицю >>>>" перехід до іншої сторінки програми.
		2.2.14 Перша радіокнопка "Усі символи" генерація пароля з усіма літерами латиниці, цифрами та спеціальними знаками.
		       Максимальна довжина пароля, якщо повторювані символи заборонені - 94.
		2.2.15 Друга радіокнопка "Тільки букви" генерація пароля тільки з латинськими буквами верхнього та нижнього регістру.
		       Максимальна довжина пароля, якщо повторювані символи заборонені - 52.
		2.2.16 Третя радіокнопка "Тільки цифри" генерація пароля тільки з цифр від 0 до 9 включно.
		       Максимальна довжина пароля, якщо повторювані символи заборонені - 10.
		2.2.17 Четверта радіокнопка "Букви і цифри" генерація пароля об'єднує в собі символи з попередніх 2 радіокнопок.
		       Максимальна довжина пароля, якщо повторювані символи заборонені - 62.
		2.2.18 П'ята радіокнопка "Букви і знаки" генерація пароля з усіма літерами латиниці та зі спеціальними знаками
		       Максимальна довжина пароля, якщо повторювані символи заборонені - 84.
		2.2.19 П'ята радіокнопка "Цифри і знаки" генерація пароля з усіма цифрами та зі спеціальними знаками
		       Максимальна довжина пароля, якщо повторювані символи заборонені - 42.

	2.3 На додатковій сторінці ви зможете побачити 8 звичайник кнопкок і таблицю, в якій можна змінювати 2 стовчика.
		2.3.1 Перша кнопка "UA" змінює мову усіх елементів поточної сторінки програми на українську, а також оновлює таблицю.
		2.3.2 Друга кнопка "☼" або "☾" змінює тему інтерфейсу поточної сторінки на світлу та темну відповідно, а також оновлює таблицю.
		2.3.3 Третя кнопка "EN" змінює мову усіх елементів поточної сторінки програми на англійську, а також оновлює таблицю.
		2.3.4 Четверта кнопка "<<<< Назад" повертає користувача на головну сторінку програми.
		2.3.5 П'ята кнопка "Оновити" оновлює вміст таблиці.
		2.3.7 Шоста кнопка "Змінити" пропонує внести зміни в базу даних, якщо хоча б одне значення опису пароля або значення пароля
		      було змінено. Якщо жодного значення не було змінено, то виводиться відповідне повідомлення. Якщо значення опису пароля 
		      повторюється більше 1 раза в стовпчику "Опис пароля", то виводиться відповідне повідомлення.
		2.3.8 Сьома кнопка "Видалити" видаляє рядок, що містить інформацію про пароль за вказаним унікальним ідентифікатором, 
		      що заходиться в першій колонці таблиці.
		2.3.9 Восьма кнопка "Вихід" завершення роботи програми. Закриття додатку.
